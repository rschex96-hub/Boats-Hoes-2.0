# Boats & Hoes simple site

This is a simple static website that you can upload to Vercel.

## Deploy on Vercel
1. Log in to Vercel
2. Click **Add New Project**
3. Drag this whole folder into Vercel
4. Click **Deploy**

## Files
- `index.html`
- `styles.css`
- `script.js`

## Notes
This is a front-end demo booking site.
It does not yet save bookings, process payments, or send confirmations.
